핵심 요구사항 분석

  • 이카운트 ERP Open API 실시간 연동 — 수출입, 재고, 회계 데이터를 배치/실시간 방식으로 BigQuery에 적재하고, API 호출 제한(Rate Limit)을 고려한 스케줄링 구현
  • GCP BigQuery 데이터 웨어하우스 — 스키마 최적화, Google Sheets 양방향 동기화, 에러 모니터링 및 알림 시스템
  • Gemini AI 무역 서류 자동 생성 — Invoice, Packing List 초안 자동 생성, 다국어 번역, 국가별 양식 템플릿 관리, Google Docs/PDF 변환
  • AppSheet 현장 입출고 앱 — 바코드 스캔/등록, BigQuery 및 ERP 실시간 연동, 별도 교육 없이 사용 가능한 직관적 UI
  • Looker Studio 경영 대시보드 — 매출/재고/수출입 현황 시각화, 주간/월간 보고서 자동 생성 및 이메일 발송, 부서별 접근 권한 제어

제안 포인트

본 프로젝트는 Google Cloud 생태계(BigQuery, Cloud Functions, AppSheet, Looker Studio, Gemini AI)를 중심으로 구축되어, 각 서비스 간 네이티브 연동이 핵심입니다. 다음과 같은 접근으로 프로젝트를 수행합니다:

  • ERP API 연동 안정성 확보 — 이카운트 ERP Open API의 인증 토큰 관리, Rate Limit 대응, 실패 시 자동 재시도 로직을 우선 구축하여 데이터 파이프라인의 안정성을 확보합니다.
  • Google 생태계 네이티브 통합 — BigQuery를 허브로 AppSheet, Looker Studio, Google Sheets가 유기적으로 연결되는 아키텍처를 설계합니다.
  • AI 서류 생성 품질 관리 — Gemini AI가 생성한 무역 서류 초안을 담당자가 검토·수정할 수 있는 워크플로우를 구성하여, AI 자동화와 사람의 검증을 균형 있게 배치합니다.
  • 단계별 마일스톤 — 1개월 내 ERP 데이터 연동, 2개월 차 AppSheet/대시보드 시연, 3개월 내 최종 오픈이라는 클라이언트 일정에 맞춘 단계별 납품을 진행합니다.

PRD (Product Requirements Document)

1. 프로젝트 배경 및 목적

50명 규모의 중소 무역회사에서 이카운트 ERP를 사용 중이나, ERP 데이터의 활용이 수작업에 의존하고 있습니다. 무역 서류(Invoice, Packing List) 작성, 입출고 관리, 경영 현황 파악 등의 업무를 자동화하여 다음 목표를 달성합니다:

  • 직원 AI 활용률 50% 이상 달성
  • 서류 작성 시간 40% 단축
  • 주간 보고서 작성 시간 8시간 → 10분 이내 단축

3. 기능 요구사항

3.1 이카운트 ERP 실시간 데이터 파이프라인

  • [필수] 이카운트 ERP Open API 연동 및 인증 토큰 관리
  • [필수] 수출입, 재고, 회계 데이터 실시간/배치 추출 및 정제
  • [필수] Google BigQuery 데이터 웨어하우스 적재 및 스키마 최적화
  • [필수] BigQuery ↔ Google Sheets 양방향 실시간 데이터 동기화
  • [필수] API 호출 실패/데이터 누락 방지를 위한 에러 모니터링 및 알림

3.2 무역 서류 AI 자동 생성

  • [필수] Gemini AI 연동 무역 서류(Invoice, Packing List) 초안 자동 생성
  • [필수] ERP 수발주 데이터 기반 필수 항목 자동 매핑 및 기입
  • [필수] Google Docs/PDF 자동 변환 및 Google Drive 폴더 분류 저장
  • [필수] 다국어 번역 지원 및 국가별 무역 서류 양식 템플릿 관리
  • [필수] 원클릭 서류 발행 UI (Google Sheets/AppSheet 연동)

3.3 AppSheet 현장 입출고 앱

  • [필수] AppSheet 기반 현장 작업자용 모바일 입출고 스캔/등록
  • [필수] 모바일 앱 입력 데이터의 BigQuery 및 이카운트 ERP 실시간 업데이트
  • [필수] 별도 교육 없이 즉시 사용 가능한 직관적 UI

3.4 Looker Studio 경영 대시보드

  • [필수] 전사 경영 지표(매출, 재고, 수출입 현황) 시각화 대시보드
  • [필수] 주간/월간 경영 보고서 자동 생성 및 이메일 정기 발송
  • [필수] 부서별/직급별 대시보드 접근 권한 제어

3.5 시스템 관리

  • [필수] 데이터 보안 설정 및 접근 권한 관리
  • [선택] 사용자 활동 로그 및 감사 추적
